# Dataset card — Synthetic Physics (template)

- **Source:** generated by `src/data/synthetic_physics.py` (deterministic, seeded).
- **Content:** short clips of simple 2D physical scenarios with exact per-frame ground-truth state.
- **Splits:** controlled via config (`num_clips`, `seed`, `scenarios`).
- **Labels:** position, velocity, acceleration, radius, mass, visibility, gravity, collision events.
- **Intended use:** validating probes/metrics and studying physical-variable manifolds.
- **Limitations:** simplified renderer/dynamics; a proxy for controlled probing, not photorealism.
